# 接口定义
```js
请求参数:
        所有请求，需要在header中带上"X-B3-Traceid",字段内容为uuid(V4)取16位长度结果
响应结果:
        {
            "code":0,                                                       // 表示成功
            "data":{},                                                      // 业务数据
            "msg":"SUCCESS",                                                // 错误详情
            "signature":null,                                               // 响应签名
            "success":true                                                  // 响应状态
        }
```
